End users use application software to accomplish specific tasks. For example, we use word processing programs to create letters, documents, and reports. However, end users also use system software. System software works with end users, application software, and computer hardware to handle the majority of technical details. For example, system software controls where a word processing program is stored in memory, how commands, are converted so that the system unit can process them, and where a completed documents or file is saved.
System software is not a single program. Rather, it is a collection or a system of programs that handle hundreds of technical details with little or no user intervention. System software consists of four types of programs.
