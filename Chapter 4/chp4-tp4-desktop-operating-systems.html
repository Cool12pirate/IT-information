Desktop OS

Every personal computer has an operating system controlling its operations. The most widely used desktop operating systems are Windows, macOS, UNIX, and Linux.

Windows

Microsoft's Windows is the most widely used personal computer operating system. Because its market share is so large, more application programs have been developed to run under Windows than any other operating system. Windows comes in a variety of different versions and is designed to run with a variety of different microprocessors. The three recent versions are Windows 7, Windows 8 and Windows 10.

‚óè Windows 7 operating system was made available by Microsoft towards the end of 2009. Since Windows Vista, the earlier operating system from Microsoft was criticised by many people, Windows 7 was designed to replace it. New libraries and file sharing capabilities were added in this operating system. Windows 7, see Figure 4-6, was well-received by users. Its performance was also found to be very good as compared to Vista.

‚óè Windows 10 was announced in 2015. It merges Windows' desktop and mobile operating systems. Unlike previous Windows versions, Windows 10, see Figure 4-7, runs on all Windows devices, including desktops, tablets, and smartphones. This unification has brought several mobile OS innovations to the desktop OS, including Contana, a digital assistant that can run on desktops, tablets, and smartphones. Other innovations include improved gaming with Xbox gaming environment, a new web browser, and support for Microsoft HoloLens, an argumented reality tool. macOS

Apple has been the leader in the development of powerful and easy-to-use personal computer operating systems since its introduction of the Macintosh personal computer in 1984. Designed to run only with Apple computer, macOS is not as widely used as Windows operating system. As a result, fewer application programs have been written for it. However, with increasing sales of Apple computers, the use of macOS has been rapidly increasing and is widely recognised as one of the most innovative operating systems. macOS X is the most widely used Mac desktop operating system. Three recent versions are mentioned below:

‚óè macOS Sierra, which was announced in 2016 and introduced several improvements, including new advanced utilities, improved integration with iOS devices, and better integration with cloud computing.

‚óè macOS High Sierra, which was announced in 2017, see Figure 4-8. It provides a new file system for faster, more secure data storage, improved video compression, and virtual reality support. The update includes improvement to Apple applications, such as the Internet browser Safari, and the photo application, Photos.

UNIX and Linux

The UNIX operating system was originally designed in the late 1960s to run on minicomputers in network environments. Over the years, UNIX has evolved with numerous different versions. Now, it is widely used by servers on the web, mainframe computers, and very powerful personal computers. There are a large number of different versions of UNIX. Linux is an operating system that extended one of the UNIX versions. It was originally developed by a graduate student at the University of Helsinki, Linus Torvalds, in 1991. He allowed free distribution of the operating system code and encouraged others to modify and further develop the code. Programs released in this way are called open source. Linux is a popular and powerful alternative to the Windows operating system, see Figure 4-9. Linux has been the basis of several other operating systems. For example, Google's Chrome OS is based on Linux. Chrome OS integrates with web servers to run applications and to perform other traditional operating system functions. This capability has made Chrome OS a popular choice for inexpensive notebook computers that use cloud computing and cloud storage that would normally require expensive hardware. One limitation of these computers is that thier efficiency is dependent upon the speed of thier Internet connection.

Virtualisation

Application programs are designed to run with particular operating systems. What if you wanted to run two or more applications each requiring a different operating system? One solution would be to install each of the operating systems on a different computer. There is, however, a way in which a single physical computer can support multiple operating systems that operate independently. This approach is called virtualisation. When a single physical computer runs a special program known as virtualisation software, it operates as though it were two or more separate and independent computers, known as virtual machines. Each virtual machine appears to the user as a separate independent computer with its own operating system, see Figure 4-10. The operating system of the physical machine is known as the host operating system. The operating system for each virtual machine is known as the guest operating system. Users can readily switch between virtual computers and programs running on them. There are several programs that create and run virtual machine that can run Windows and Windows app from within the macOS itself.
