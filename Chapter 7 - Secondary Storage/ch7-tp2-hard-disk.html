Hard Disk

Hard disks save files by altering the magnetic charges of the disk's surface to represent 1s and 0s. Hard disks retrieve data and programs by reading these charges from the magnetic disc. Characters are represented by positive (+) and negative (-) charges using the ASCII, EBCDIC, or Unicode binary codes. For example, the letter A would require a series of 8 charges, see Figure 7-2. Density refers to how tightly these charges can be packed next to one another on disk. Hard disks use rigid metalic platters that are stacked one on top of another. Hard disks store and organise files using tracks, sectors, and cylinders. Tracks are rings of concentratic circles on the platter. Each track is divided into invisible wedge-shaped sections called sectors, see Figure 7-3. A cylinder runs through each track of a stack of platters. Cylinders are necessary to differentiate files stored on the same track and sector of different platters. When a hard disk formatted tracks, sectors, and cylinders are assigned. Hard disks are sensitive instruments. Thier read/write heads are surrounded by air which is about 0.000001 inch thick. It is so thin that a smoke particle, fingerprint dust, or human hair could case what is known as a head crash, see Figure 7-4. A Head crash occurs when a read-write head makes contact with the hard disk's surface or with particles on its surface. A head crash is a disaster for a hard disk. The disc surface is scratched, and some or all of the data is destroyed. At one time, head crashes were commonplace. Now fortunately, they are rare. There are two basic types of hard disks: internal and external.

Internal Hard Disk

An Internal hard disk is located inside the system unit. There hard disks are able to store and retrieve large quantities of information quickly. They are used to store programs and data files. For example, nearly every personal computer uses its internal hard disk to store its operating system and major applications such as Word and Excel. To ensure adequate performance of your internal hard disk and the safety of your data, you should perform routine maintenance and periodically make backup copies of all important files. For hard disk maintenance and backup procedures. refer to the section on Windows utilities Disc Cleanup and Optimise Drivers in the book.

External Hard Drives

While internal hard disks provide faster acess. They have a fixed amount of storage and cannot be easily removed from the system unit. External hard disks provide slower access and are typically connected to a USB for Thunderblot port on the system unit can be easily removed. Like an internal hard disk, external hard disks have a fixed amount of storage. However, since each removable hard disk can be easily replaced by another removable hard disk, a single port on the system unit can provide access to an unlimited amount of storage, see Figure 7-5. External hard drives use the same basic technology as internal hard disks and are used primarily to complement an internal hard disk. Because they are removable, they are particularly useful to protect or secure sensitive information. Other uses external drives include backing up the contents of the internal hard disk and providing additional hard disk capacity.

Performance Enhancements

Three ways to improve teh performance of hard disks are disc caching, redundant arrays of inexpensive disc, and file compression/decompression. Disc caching improves hard disk performance by anticipating data needs. It performs a function similar to cache memory discussed in ....... . While cache memory improves processing by acting as a temporary high-speed holding area between memory and the CPU, disc caching improves processing by acting as a temporary high-speed holding area between a secondary storage device and the CPU. Disc caching requires a combination of hardware and software. During idle processing time, frequently used data is automatically identified and read from the hard disk into the disk cache. When needed, the data is then accessed directly from memory. The transfer rate from memory is much faster than from the hard disk. As a result, overall system performance is often increased by as much as 30 percent.

Redundant Arrays of Inexpensive Disks (RAID) improve performance by expanding external storage, improving access speed, and providing reliable storage. Several inexpensive hard disk drives are connected to one another. These connections can be by a network or within specialised RAID devices, see Figure 7-6. The connected hard disk drives are related or grouped together, and the computer system interacts with the RAID system as though it were a single large-capacity hard disk drive. The result is expanded storage capability, fast access speed, and high reliability. For these reasons, RAID is often used by Internet servers and large organisations.

File compression and file decompression increase storage capacity by reducing the amount of space required to store data and programs. File compression is not limited to hard disk systems. It is frequently used to compress files on DVDs, CDs, and flash drives as well. File compression also helps to speed up transmission of files from one computer system to another. Sending and receiving compressed files across the Internet is a common activity. File compression programs scan files for ways to reduce the amount of required storage. One way is to search for repeating patterns. The repeating patterns are replaced with a token, leaving enough tokens so that the original can be rebuilt or decompressed. These programs often shrink files to a quarter of thier original size. Windows and Mac operating systems provide compression and decompression utilities. For more advanced compression schemes, you can use specialised utility such as WinZip. For a summary of performance enhancement techniques, refer Table 7-1.

Privacy
Diminishing secondary storage prices have an unexpected impact on privacy. The availability of cheap digital storage has resulted in a permanent digital record of our lives available for all to see on the Internet. Once an image, video, or message is released on the Internet, it is very difficult very difficult to remove. Some argue that we all have a "right to be forgotten" and that major Internet compaines like Facebook and Google should help people permanently remove records of embarrassing or unpleasant moments. Tips
Does your internal hard disk run a lot and seem slow? The problem could be with fragmented files-files, when saved were broken into pieces (fragments) and stored in different locations on your hard disk, which take longer for the hard disk to access. Defragging rearranges the file parts so they stored in adjacent locations. To clean up the disk and speed up access times, consider defragmenting.

1. Start Disk Defragmenter: Type disk defrag into the Start menu search box and press Enter to open the Disk Defragmenter utility. Click Defragment disk to begin defragmenting your disk.
2. Keep working: You can contnue running other applications while your disk is being defragmented. But, then, your computer operates more slowly, and Disk Defragmenter takes longer to finish.
3. Automate: Windows will defragments your disk for you automatically. Click Configure schedule in the Disk Defragmenter utility to set this to a time that is convenient for you.
