Mass Storage Devices

It is natural to think of secondary storage media and devices as they relate to use an individuals. It may not be as obvious how important these matters are to organisations. Mass storage refers to the tremendous amount of secondary storage required by large organisations. Mass storage devices are specialised high-capacity secondary storage devices designed to meet organisational demands for data storage. These mass storage solutions allow large corporations and institutions to centralise thier maintenance and security of data, thereby reducing costs and personal.

Enterprise Storage System

Storage Area Network

A recent mass storage development in storage area network (SAN) systems. SAN is an architecture to link remote computer storage devices, such as enterprise storage systems, to computers such that the devices are as available as locally attached drives. In a SAN systems, the user's computer provide the file system for storing data, but the SAN provides the disk space for data. The key to a SAN is a high-speed network, connecting individual computers to mass storage devices. Special file systems prevent simultaneous users fro interfering with each other. SANs provide the ability to house data in remote locations and still allow efficient and secure access.
