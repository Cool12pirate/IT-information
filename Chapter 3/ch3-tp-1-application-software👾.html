Not long ago, trained specialists were required to perform many of the operations you can now do with a personal computer. Market analyse used calculators to project sales. Graphic artists created designs by hand. Data processing clerks created electronic files to be stored on large computers. But now you can do all these tasks-and many others-with personal computer and appropriate application software.
There are two kinds of software. System software works with end user, application software, and computer hardware to handle the majority of technical details. Application software can be described as end user software and is used to accomplish a variety of tasks.
Application software can further be divided into three categories. One category, general-purpose applications, include word processing programs, spreadsheets, presentation graphics packages, etc. Another category, specialised applications, include thousands of other programs that tend to be more narrowly focused and used in specific disciplines and occupations. For exmaple, designers and advertising agencies use CorelDraw, Photoshop, GIMP, Inkscape, InDesign, etc. while NetSuite ERP, Zoha books, Busy, Marg, Saral and Tally, etc. are accounting softwares. The third category, mobile apps, comprises add-on features on programs typically designed for smartphones and tablets. These apps can be general purpose or specialised and range from general purpose images editing apps like Snapseed, TouchRetouch, Enlight, Pixlr to health monitoring apps like Google Fit, Pacer, Pedometer, runkeeper, etc. There are also specialised apps like VoLT, PowerVocab, A Word A DayWidget, for vocabulary learning; netbanking apps like pocket, maps and navigation apps like Google Maps and Productivity apps like, PhoneClean, Google drive.

A user interface is that portion of the application which allows you to control and interact with the program. Depending on the application, you can use a mouse, a pointer, a keyboard, and/or your voice to communicate with the application. Most general-purpose applications use a mouse and a Graphical User Interface (GUI) that displays graphical elements called icons to represent familiar objects. The mouse controls a pointer on the screen that is used to select items such a icons. Nowadays you can also use devices such as microphone to give voice instructions to computers. Another feature is the use of windiws to display information. A window is simply a rectangular area that can contain a document, program, or message. (Do not confuse the term windows with the various versions of Microsoft's Windows operating system, which is a program.) More than one window can be opened and displayed on the computer screen at one time.
Traditionally, most software programs use a system of menus, toolbars, abd dialog boxes, see Figure 3-1.

  • Menus present commands that are typically displayed in a menu bar at the top of the screen.
  • Toolbars typically appear below the menu bar and include small graphic elements called buttons that provide shortcuts for quick access to commonly used commands. These can be multiple toolbars is an application, e.g., picture toolbar and text toolbar.
  • Dialog boxes provide additional information and request user input.

    • Figure 3-1 Traditional Graphical user Interface

    Many applications, and Microsoft applications in particular, use an interface known as the Ribbon GUI to make it easier to find and use all the features of an application; this GUI uses a system of ribbons, tabs, and galleries, see Figure 3-2.

  • Ribbons replace menus and toolbars by organising commonly used commands into a set of tabs. These tabs display command buttons that are most relevant to the tasks being performed by the user.
  • Tabs are used to divide the ribbon into major activity areas. Each tab is then organised into groups that contain related items. Some tabs, called contextual tabs, appear only when they are needed and anticipate the next operation to be performed by the user.

    • Figure 3-2 Ribbon GUI

  • Galleries simplify the process of making a selection from a list of alternatives. This is accomplished by graphically displaying the effect of alternatives before being selected.

    Most applications provide a variety of features to make entering/presenting, editing and formatting documents easy. Some of the most common features include:

  • Spell checker: Looks for misspelled words.
  • Alignment: Align numbers, characters and pictures to the centre, left to right side of the page.
  • Fonts and font sizes (perhaps using character effect): Specify the size and style of entered numbers and text.
  • Character effects: Provide a variety of different typefaces, such as bold or italics.
  • Edit options: Provide easy ways to edit text, such as cut, copy, and paste.
