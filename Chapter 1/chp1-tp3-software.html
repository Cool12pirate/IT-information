Introduction to Information Technology

● SOFTWARE

Software, as we mentioned, is another name for a program. A program is a set of instructions that tell a computer how to process data into the form you want. In most cases, the words software and program are interchangeable. There are two major kinds of software: system software and application software. Application software (apps) are created for specific use of end users. System software are designed to run computers.

⊶ System Software ⊷

The user interacts with application software. System software enables the application software to interact with the computer hardware. System software is 'background' software that helps the computer manage its own internal resources.
System software is not a single program. Rather, it is a collection of programs, including the following:

  • Operating System are program that coordinate computer resources, provide an interface between users and the computer, and run applications. Smartphones, tablets, and many other mobile devices are embedded operating systems, also known as real-time operating system (RTOS). Desktop computers use stand-alone operating systems like Windows 10 and macOS, see Figure 1-2 and 1-3. Networks use network operating systems (NOS).
  • Utilities perform specific tasks related to managing computer resources. One of the most essential utility program that every computer system should have is an antivirus program. This program protects your computer system from viruses or malicious programs that are all too often deposited onto your computer from the Internet. This program can damage software and hardware, as well as compromise with security and privacy of your personal data. If your computer does not have an antivirus program installed on it, you need to get one. To see how you can install a free antivirus program on your computer, see Making IT work for You: Free Antivirus Program .
  • Device drivers are specialised programs designed to allow particular input or output devices to communicate with the rest of the computer system.

    • ⊶ Application Software ⊷

    Application software might be described as end-user software. Three types of application software are general-purpose, specialised, and mobile applications.

  • General-purpose applications are widely used in nearly all career areas. They are the kinds of program you have to know to be considered an efficient and effective end user. Some of the best known general-purpose applications are listed in Table 1-2.
    • Table 1-2 General-purpose Applications
    Type Description
    File explorer Program to manage files and folders
    Browser Connect to websites and display web pages
    Word processor Prepare written documents
    Spreadsheets Analyse and summarise numerical data
    Database management systems Organise and manage data and information
    Presentation graphics Communicate a message or persuade other people
    Email client Application to send, receive and read emails.
  • Specialised applications include thousands of other programs that are more narrowly focused on specific disciplines and occupations. Two of the best knwon specilaised applications are graphics and web authoring programs.
  • Mobile apps, also knwon as mobile applications ot simply apps, are small program primarily designed for mobile devices such as smartphones and tablet computers, see figure 1-4. As of 2019, there are over 3 million apps. The most popular mobile apps are used for social networking, playing games, and downloading music and videos.
